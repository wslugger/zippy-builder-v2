---
description: Code with /tdd (Test Driven Development)
---

1. Understand the requirement or bug description.
2. Read `.agent/rules/TESTING_SUITE.md`.
3. Create a failing test in the appropriate `__tests__/` directory.
// turbo
4. Run the test to confirm it fails: `npm run test`
5. Implement the code to make the test pass.
// turbo
6. Run the test again to verify the fix: `npm run test`
